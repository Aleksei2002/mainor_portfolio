The rStudio website is created to introduce yoga training sessions. A convenient menu allows users to quickly and easily register for a suitable training group by choosing the best time.

SEO

rStudio - Jooga Stuudio

Landing Page

Joga meeleolu paremaks edasindmiseks, veebilehekülje olid võetud rahulikud rahulikud värvid.

Client

The client requested a minimalist-style website, richly illustrated with yoga images. They also asked to create a booking system that would allow studio clients to register for yoga classes.

Goal

Since the website was intended not only to introduce yoga but also to support its practice, it was necessary to create an intuitive and simple way for users to register for a class or cancel their booking.
